Code highlight

Code highlight is a feature with predefined styles for HTML, JavaScript and JSON. These styles are a customization for the languages supported by the library highlight.js.

You do not need the entire template to work with code highlight. You only need the highlight.js library and the admin4b.highlight.min.css present in the NPM package.
